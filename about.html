About Neon Dash

Neon Dash is a browser-based arcade game created by Yash Rastogi. The goal of this project is to build simple, engaging, and fast-loading web games using modern web technologies like HTML, CSS, and JavaScript.

This website is not just about playing a game — it is also an experiment in building interactive web experiences and improving user engagement through minimal design and smooth gameplay.

We aim to provide users with a fun and distraction-free gaming experience that works across both desktop and mobile devices without requiring any downloads.

In the future, more games and learning-based content will be added to this platform.
